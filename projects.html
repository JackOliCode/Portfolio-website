MyFlix Projects

Above are the three projects I made for the MyFlix app assignment. I began with creating a database using JSON and Node.js and the database is hosted on MongoDB.

After this, I then made a single page app in React, using multiple components to allow the user to view a range of pre-existing movies, their own user profile and the movies that they have added to their favourites. The front end app syncs with the MyFlix API mentioned above.

Finally, after some time away from the project, I returned to reimagine the front end client and used Angular to make it on this occasion.
